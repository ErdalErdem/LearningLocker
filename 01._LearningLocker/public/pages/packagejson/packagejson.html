package-json

This is our package-json The file can be divided into three: meta info, scripts, and dependencies.

Meta: tells us information about the author, license and version and this is important if your programs needs to be shared online.

Dependencies: tells us what we need to import to run our program

Script: allows running longer terminal commands, stored in a key-value pair. 


    "dependencies": {
        "cross-env": "^7.0.3",
        "express": "^4.18.2"
      },
      "name": "Learninglocker",
      "version": "1.0.0",
      "main": "app.js",
      "type": "module",
      "scripts": {
        "test": "echo \"Error: no test specified\" && exit 1",
        "start:dev": "cross-env PORT=8080 nodemon app.js",
        "start:prod": "cross-env PORT=80 node app.js"
      },
      "keywords": [],
      "author": "",
      "license": "MIT",
      "description": ""
    }
