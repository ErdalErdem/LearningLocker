Basic Commands for VSCode and Node.js

VSCode Commands

code . - Open the current directory in VSCode.

Ctrl + ` - Open integrated terminal.

Ctrl + P - Quick open files.

Ctrl + Shift + P - Open command palette.

Ctrl + B - Toggle sidebar visibility.

Node.js Commands

node app.js - Run a Node.js application.

npm init - Initialize a new Node.js project.

npm install package_name - Install a package.

npm start - Run the script defined as "start" in package.json.

npm run script_name - Run a script defined in package.json.
